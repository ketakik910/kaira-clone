# ✨ Kaira UI Clone (React + GSAP)

## 📌 Overview

This project is a fully responsive, animated frontend clone inspired by the Kaira fashion website. It was built from scratch to practice modern frontend development, focusing on smooth animations, clean UI design, and scalable component architecture using React.

---

## 🚀 Tech Stack

* **React (Vite)**
* **GSAP** – for advanced animations
* **Swiper** – for interactive sliders
* **CSS3** – for styling and responsiveness

---

## ✨ Features

* Smooth, scroll-based animations powered by GSAP
* Interactive and responsive sliders using Swiper
* Fully responsive design (mobile, tablet, desktop)
* Clean and modern UI layout
* Reusable and modular React components

---

## 📂 Project Structure

The project follows a component-based architecture:

* `Nav` – Navigation bar
* `Page1 - Page10` – Individual UI sections
* `Footer` – Footer section
* Separate CSS files for each component
